Tales of MeatMan

Little 2D platformer made during a game course from Jason Weimann. You can jump, double jump, wall jump, bump snails and shells. There is a single really simple level at the moment, but I may create more in the future !

Input
Description
Q-D / Arrows
Move character
Space

Grimwatch

Top down Hack&Slash with local coop (1 to 4 players). Once in the HUB (the village), there is a purple circle that teleports you to another scene where you can fight Orc NPCs. Still needs a lot of tweaking / polishing (especially camera).

Input
Description
ZQSD / WASD / Arrows / Gamepad left stick
Move character
Mouse / Gamepad right stick
Move camera
Space / Gamepad south button
Jump
Left mouse button / Game west button
Press : Light attack / Hold : Heavy attack
